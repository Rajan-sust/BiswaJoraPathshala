### BiswaJora

[Setup a Laravel Project Cloned from Github.com](https://goo.gl/AJcJRq)

```
git clone https://github.com/Rajan-sust/BiswaJoraPathshala
cd BiswaJora-Pathshala
composer install
npm install
cp .env.example .env
php artisan key:generate
```

Edit the .env file

```
php artisan migrate
```
